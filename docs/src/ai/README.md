## AI编程

> 用AI工具加速PB开发，让老技术焕发新生。

### 为什么用AI辅助开发？

| 场景 | 传统方式 | AI辅助 |
|------|----------|--------|
| 写代码 | 手敲每行代码 | 描述需求，AI生成 |
| 改bug | 查文档、问人 | AI分析原因给方案 |
| 学新知识 | 查资料、看视频 | AI直接讲解 |
| 写文档 | 手动编写 | AI自动生成 |

### 推荐工具

| 工具 | 特点 | 推荐指数 |
|------|------|:--------:|
| **Cursor** | IDE集成，上下文理解强，支持中文 | ⭐⭐⭐⭐⭐ |
| **Claude Code** | 命令行工具，代码能力强 | ⭐⭐⭐⭐ |
| **ChatGPT** | 通用对话，快速问答 | ⭐⭐⭐ |

### 实战应用

#### 用AI生成PB代码

```
提示词：帮我写一个PB数据窗口的查询功能，
支持按员工姓名模糊查询，按部门精确查询
```

#### 用AI解释代码

```
提示词：这段PB代码是什么意思？帮我解释一下
[粘贴代码]
```

#### 用AI优化代码

```
提示词：这段PB代码性能不好，帮我优化一下
[粘贴代码]
```

#### 用AI写注释和文档

```
提示词：帮这段PB代码添加注释，并生成使用文档
[粘贴代码]
```

### 使用技巧

1. **提示词要具体** — 描述清楚需求，AI回答更准确
2. **提供上下文** — 给AI足够的背景信息
3. **迭代优化** — 不满意就让AI重新生成
4. **人工审核** — AI生成的代码需要检查和测试

### 注意事项

- ⚠️ 敏感信息不要发给AI
- ⚠️ AI生成的代码需要人工审核
- ⚠️ 保持学习，不要过度依赖

---

> AI是工具，不是替代。用好AI，让开发效率翻倍。
