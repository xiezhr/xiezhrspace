## Git/GitHub

> 版本控制是团队协作的基础，也是开源项目的必备技能。

### 为什么学Git？

- 代码版本管理，随时回退
- 团队协作开发必备
- GitHub 开源项目参与
- 面试必问技能

### 学习内容

| 主题 | 说明 |
|------|------|
| Git基础 | 安装、配置、基本命令 |
| 分支管理 | 创建、合并、解决冲突 |
| GitHub使用 | 仓库管理、PR、协作 |
| 团队协作 | Git Flow、代码评审 |

### 常用命令

```bash
# 初始化仓库
git init

# 克隆仓库
git clone <url>

# 查看状态
git status

# 添加文件
git add .

# 提交
git commit -m "message"

# 推送
git push origin main

# 拉取
git pull origin main
```
