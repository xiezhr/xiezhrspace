## Git / GitHub 

> 版本控制是团队协作的基础，也是开源项目的必备技能。

### 为什么学 Git？

- **代码版本管理**，随时回退与追溯
- **团队协作开发**必备工具
- **GitHub / GitLab** 开源与企业项目参与
- **面试必问**技能，职业发展刚需

### 常用命令速查

```bash
# 初始化仓库
git init

# 克隆仓库
git clone <url>

# 查看状态
git status

# 添加文件
git add .

# 提交
git commit -m "message"

# 推送
git push origin main

# 拉取
git pull origin main

# 创建并切换分支
git checkout -b feature-branch

# 合并分支
git merge feature-branch
```
