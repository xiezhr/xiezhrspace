## 项目导入与结构

> 打开项目、模块、依赖管理。

### 项目导入方式

| 方式 | 适用场景 |
|------|---------|
| Open | 打开已有项目 |
| Import Project | 导入 Eclipse/Gradle 项目 |
| Get from VCS | 从 Git 克隆 |

### 项目结构

- Project（项目）
  - Module（模块）
    - Source Root（源码目录）
    - Resource Root（资源目录）
