## IDEA使用

> IntelliJ IDEA 是 Java 开发的最佳 IDE，提升开发效率的神器。

### 为什么学IDEA？

- Java/SpringBoot 开发的首选 IDE
- 智能代码提示，开发效率翻倍
- 强大的调试功能，快速定位问题
- 丰富的插件生态

### 学习内容

| 主题 | 说明 |
|------|------|
| 安装与配置 | 下载、安装、基础设置 |
| 常用快捷键 | 代码编辑、导航、重构 |
| 调试技巧 | 断点、变量查看、条件断点 |
| 实用插件 | 推荐提升效率的插件 |

### 常用快捷键

| 功能 | Windows | Mac |
|------|---------|-----|
| 查找类 | Ctrl+N | Cmd+O |
| 查找文件 | Ctrl+Shift+N | Cmd+Shift+O |
| 全局搜索 | Ctrl+Shift+F | Cmd+Shift+F |
| 格式化代码 | Ctrl+Alt+L | Cmd+Option+L |
