## IDEA使用

> IntelliJ IDEA 是 Java 开发的最佳 IDE，提升开发效率的神器。

### 为什么学IDEA？

- Java/SpringBoot 开发的首选 IDE
- 智能代码提示，开发效率翻倍
- 强大的调试功能，快速定位问题
- 丰富的插件生态

### 学习内容

| 主题 | 说明 |
|------|------|
| 基础配置 | 下载安装、初始配置、项目结构、设置同步 |
| 代码编辑与导航 | 快捷键、代码补全、重构、多光标编辑 |
| 调试与测试 | 断点调试、远程调试、单元测试 |
| 版本控制 | Git 集成、代码审查、GitHub 协作 |
| 插件与效率工具 | 必备插件、代码质量、数据库工具 |
| 进阶技巧 | 多模块项目、HTTP Client、性能优化 |

### 常用快捷键

| 功能 | Windows | Mac |
|------|---------|-----|
| 查找类 | Ctrl+N | Cmd+O |
| 查找文件 | Ctrl+Shift+N | Cmd+Shift+O |
| 全局搜索 | Ctrl+Shift+F | Cmd+Shift+F |
| 格式化代码 | Ctrl+Alt+L | Cmd+Option+L |
