## Nginx

> Nginx 是高性能的 Web 服务器与反向代理，是后端开发和运维必备技能。

### 为什么学 Nginx？

- **高性能反向代理**，支撑高并发流量
- **负载均衡**，保障服务高可用
- **静态资源服务**，加速前端部署
- **生产环境标配**，运维与后端必会
