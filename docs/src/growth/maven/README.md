## Maven

> Maven 是 Java 生态最主流的项目构建与依赖管理工具，掌握它是高效开发的前提。

### 为什么学 Maven？

- **依赖管理**，告别手动下载 jar 包
- **标准化构建**，统一项目结构与生命周期
- **多模块管理**，应对大型项目架构
- **企业开发标配**，面试与工作必备
