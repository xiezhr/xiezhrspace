## 技术成长

> PB开发者的技能扩展之路，为职业发展铺路。

### 为什么需要技术成长？

PB开发者在工作中不仅需要掌握PB本身，还需要：

- **开发工具** — IDEA、Git 提升开发效率
- **运维基础** — Linux 命令用于服务器管理
- **技术转型** — SpringBoot+Vue 拓展职业道路

### 学习内容

| 模块 | 内容 | 适用场景 |
|------|------|----------|
| [IDEA使用](idea/) | 快捷键、插件、调试技巧 | 提升开发效率 |
| [Git/GitHub](git/) | 版本控制、团队协作 | 代码管理必备 |
| [Linux核心命令](linux/) | 常用命令、Shell脚本 | 服务器运维 |
| [SpringBoot+Vue](springboot-vue/) | 前后端分离开发 | 技术转型 |

### 学习建议

1. **先学工具** — IDEA、Git 立即能用，提升效率
2. **再学运维** — Linux 命令用于服务器管理
3. **最后学转型** — SpringBoot+Vue 为职业发展做准备
