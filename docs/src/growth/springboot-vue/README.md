## SpringBoot+Vue

> PB开发者的技术栈转型指南，从桌面应用到前后端分离。

### 为什么学SpringBoot+Vue？

| 对比 | PB | SpringBoot+Vue |
|------|-----|----------------|
| 架构 | 桌面应用 | 前后端分离 |
| 部署 | 客户端安装 | 浏览器访问 |
| 扩展 | 单机为主 | 分布式、微服务 |
| 市场 | 存量维护 | 主流开发模式 |

### 学习路线

```
第一阶段：Java基础（2-3周）
├── Java语法
├── 面向对象
└── 集合、异常

第二阶段：SpringBoot（3-4周）
├── Spring基础
├── SpringBoot快速开发
└── MyBatis操作数据库

第三阶段：Vue前端（3-4周）
├── HTML/CSS/JS基础
├── Vue3语法
└── Element Plus组件

第四阶段：实战项目（4-6周）
├── 前后端联调
├── 登录认证
└── CRUD操作
```

### 对比学习

| 功能 | PB实现 | SpringBoot+Vue实现 |
|------|--------|-------------------|
| 数据查询 | DataWindow | MyBatis + API |
| 界面开发 | Window Painter | Vue组件 |
| 事件处理 | 事件脚本 | 前端事件绑定 |
| 数据库连接 | Transaction对象 | DataSource配置 |
