## Docker

> 容器化是现代应用部署的标准方式，Docker 是每个开发者必须掌握的工具。

### 为什么学 Docker？

- **环境一致性**，告别"在我机器上能跑"
- **快速部署**，秒级启动应用
- **微服务架构**基础设施
- **CI/CD 流水线**必备技能
